Windows 8是个烂到渣的系统

发表于 2012-08-30 | 更新于 2018-04-25 | 分类于 科技

拖了2个多礼拜，没人看嘛~就一点打字的欲望都没有 唉。。

这是我第一篇完全用Markdown写的，比用富文本编辑器舒服多了。

从Windows8的泄漏版开始就一直断断续续的在用，到目前为止所有泄漏、公开版本都用过了，终于在用Windows 8 RTM版后2天忍不住了，换回Windows 7。其实这算半个标题党，相比较Windows XP，Windows 8是天 Windows XP是地，相比较Windows 7 Windows 8就是渣。还是先夸上两句吧。

优点

任务管理器

就是那个Ctrl+Shift+ESC的那个东西，在Windows 8中明显得到了改善，简洁模式更方便使用，详细模式功能更加强大，分类清晰。

资源管理器

就是Windows XP-的「我的电脑」或者是Windows Vista+的「计算机」。新界面功能更强大，复制粘贴的那个框非常直观好用,过程中的那个进度也是。

输入法

Windows 8内置的输入法相当赞，无论是中文和英文，特别是英文，完全可以用惊艳来形容，几乎所有你能想到的非自创非完全小众的都能识别，而且还有自动大小写，非常棒。就是功能太少，无法自定义标点、词句等。总之词句识别是我用过的输入法中最好的，功能略薄弱。不愧微软收集了什么多年的用户操作习惯和隐私。

登陆密码

登录密码可以用图片来替代，即选择一张图，在图上标记几个手势，就能完成密码设定。很好用。

主题

从Windows Vista时代出现，到Windows 7时代到达巅峰的透明风格（AERO）终结了，虽然一大票人在骂，但是在体验Windows 8 RTM后发现比Windows 7的透明主题好看多了。Modern风格也非常棒，Windows 8 RTM下磁铁好看多了。

各平台统一

现在PC、笔记本、平板、手机风格、应用都统一了。平板和手机上的应用会爆发性增长，说不定能和App Store有的一拼。

优点基本也就这几个了，下面就是喷。

缺点

性能

从Windows泄漏开始就有一大帮「专家」在那说：

Windows 8会是下一个Windows Vista，因为颠覆了用户使用习惯。

Windows Vista是因为颠覆用户习惯才「失败」的么？先不论Windows Vista到底有没有失败。Windows Vista为什么「失败」？是因为发布前期硬件配置普遍低下，外加使用了全新的内核导致不少软件都出现了兼容性问题，发布间隔太长，在某些落后的国家用户无法习惯除了XP以外的系统。

我觉得Windows 8第二大缺点就是性能更差了。

我不知道微软养了这么多工程师为什么会导致新版反而性能更低下，微软官方给了一张列表，说明Windows 8性能有多大的提升，但是，我真的感觉不到。

打开一个我放置安装文件的目录，系统会卡上那么几秒，过这段时间每个文件的图标才会慢慢显示出来，这个问题在Windows 7下只有第一次打开才会出现，但是在Windows 8下是每次都会有的，即使你打开了，往下拖动，会继续卡，我不知道微软在干吗。（这个可能是权限的问题，把UAC关闭，即等于Windows7下UAC最低的时候，是不会卡的）

删除/更改文件后，地址栏的会有一条进度，重新刷新缓存，过了进10秒钟才会刷新列表，这个现象不是每次都会有，但是在频繁的更改、删除文件时出现频率较高。-

易用性

如果要用一句话说明为什么Windows 8是个渣，那就是

把本来一步就能完成的事情用三步去做。

如果你不能理解的话，以关机为例

Windows 7
  1. 按Win键
  2. 移动鼠标
  3. 点击关机键
Windows 8
  1. 鼠标移动到右下角
  2. 等待侧边出现
  3. 向上移动鼠标
  4. 点击选项
  5. 移动鼠标到电源按钮
  6. 单击
  7. 向上移动鼠标
  8. 点击关机键

别跟我钻牛角尖说按电源键，这里是不用快捷键的情况下关机动作。下面几条基本还是属于易用性和性能的。

权限混乱

UAC关不掉。自己新建的文件夹无法重命名、无法删除，这里是有较多我个人因素在里面，我在Windows Vista的时候就有的遗留问题，给好几个文件夹所有者赋权限最终导致什么都删不掉，Windows 7下因为UAC可以完全关闭就没有这个问题。主要是这几个：

Administrator、Pc Administrator、PC USER、System、Windows Live。

格式关联混乱

常见格式默认都是用Modern APP打开的，好几个视频格式会用自带的图片工具打开，关联以后经常会有关联图标丢失、关联丢失的问题。

自带的那些App不给力

比如照片、影片等App并不好用，功能非常少，甚至比之前的WMP更少。而且经常会有一些莫名其妙的问题，我在照片里进去访问「我的图片」文件夹，很多文件夹明明有图片，缩略图却显示没图片，点进去有的能检测到有的没有，还经常会缺图，明明有100张，却提示你几张等。

取消开始按钮

全部加到Modern屏幕的磁贴里（就是那个颜色方块）

也不是全部复杂化了，还是有优点的，比如原先开始列表的列表有10个，你需要点超过这10个程序的时候就得深入列表。现在按Win键就能直接点，但是本来开始菜单中这么多快捷方式一股脑全都放到这个磁块里就会有很大问题，很多无用内容也被加进来了，比如程序主页、卸载程序。如果你跟我一样喜欢装各种软件，就会发现每两天Modern屏幕上都是密密麻麻的磁贴了，想找个自己想要的并不容易。

取消开始按钮带来个很麻烦的问题就是本来Windows 7开始菜单右侧有一排功能按钮，程序里也有附件之类的，现在全部去除后点击超麻烦，要使用各种曲线救国。

原来进控制面板很简单，现在则需要先打开一个资源管理器在从地址栏进去。

打开系统更新窗口，本来开始菜单进去很简单的，现在需要比上面关机还要复杂的步骤才行。

还有管理工具等，太多了，当然你要是背出来，直接在运行里输入我也没办法。

在Windows Phone 7开始出现的磁铁在非触摸设备上是很不方便的，微软为了要统一各平台执意要牺牲现有的最大用户群——PC用户的体验，再加上Windows 8拙劣的设计，所以Windows 8在PC平台上会失败。
