Windows下视频播放器

发表于 2013-03-02 | 更新于 2018-04-26 | 分类于 科技 | 阅读次数：

我个人比较喜欢折腾各种软件，写一下我对Windows平台的视频播放器的看法。顺序是按照我记忆中自己用的顺序排的。

#Windows Media Player
我最早接触的是 WMP，就是 Windows 自带的那个。这个播放器怎么说呢，既强大又非常弱。强大的是他的管理功能，可以管理计算机上的音乐、视频和图片，我至今还没找到一款功能相近的软件。问题也非常严重，支持的格式非常少，作为播放器的功能可以说是过于简陋了，没有快捷键设置，对视频播放的设置也太少。
所以，即使他管理功能是如此的强大，并且我也对这个功能又非常大的需求，又没有任何软件能替代，但是因为这俩缺点（主要是第二个），所以被我无情的抛弃了。

#RealPlayer
当年为了看 RM 格式下的，这个软件除了能播放自己的专有格式外一无是处。臃肿程度居然能胜过 iTunes。。各种难用。

#金山影霸
几乎完全没印象，只记得这软件安装要好久好久，第一次启动有个动画。不怎么样。

#豪杰解霸
我对这个软件的印象就是能播放非常多的格式，界面非常复杂。后来被暴风影音收购了。

#暴风影音
修改自 MPC，记得汉化修改的人叫铃风什么的，我还曾经花了好几个钟头来搜索这个人来着。自从某版本（3.1？）商业化开始就变成一个大流氓，各种后台、自启动和铺天盖地的广告，当然最著名的要属当年的断网事件，神作啊 233 在耍流氓后不久我就卸了，这货后来越来越烂。唉

#快播
呵呵，你懂的。不过我也就刚开始一两个月用过，但是因为流氓和画质差就弃了。

#QQ影音
曾经是国产中最好的，但是这个团队也太。。在官方平台提建议、意见和 BUG 从来不回，再加上常年不更新，就弃了。如果你对软件没有什么要求（说白了不具备挑选软件能力），还是不错的，那些小工具都挺实用的，还整合了旋风的离线，如果你有的话。

#射手影音
唉，可惜了啊。国产开源播放器，主打字幕功能，原来挺不错的，但是在 3.6 版本自费武功，搞了一个精简版的设置（我对于软件的要求是新人能很快上手，有能力的能自定义，所以播放器我用 Firefox，我没办法用 Linux，因为我根本没办法上手）我不知道过了 3 年了，射手那帮人发现自己错了吗？如果他们还认为是QQ影音的问题的话，我只能说死了活该。
本来还留着下字幕，现在也都直接网页下载字幕了。

#KMPlayer
当年我从射手转来用的时候发现，这他瞄的就是两个极端，一个是极简一个则是极复杂。打开设置（我在之前提过，我用软件，第一件事就是打开设置框），密密麻麻的全都是选项，一下子让我觉得这个软件没法用，事实上默认设置对于我来说也的确是没法用。就几乎再也没用过，第一印象非常重要。还有一个很大的原因是KMPlayer作者跳槽去别的公司做播放器了。

#MPlayer
好像是 Linux 上最牛逼的播放器，但是 Windows 下问题很多，卡顿严重，就是播放视频的时候打开菜单时候视频会卡住，声音回一直循环那 1 秒的声音，极其蛋疼。还有个问题就是设置太复杂并且只能复杂，对我来说这也是致命伤，不过这个国人开发者比射手那帮「专业的」聪明得多，后来的版本里加入了简单设置。这就是智商上不可逾越的差距，射手影音为了照顾新手把高级设置彻底删了，MPlayer 是另加入了一个简单设置。但是还是各种用不惯。

#MPC
光给你看界面，你会有一种「上世纪软件」的感觉。对，界面非常朴素，打开设置框，内容中等，只要改一下几个快捷键就很容易上手，而且扩展性极强。但是我还是各种用不惯，很大的原因是快捷键太复杂，没有把常用的提取出来，而且快捷键设置很麻烦。最大的问题是界面（这我会在后面专门说）。

#MPC-BE
MPC 的修改版，做了很多人性化改动，但是依然用不惯。

#VLC
跨平台开源播放器，这个播放器是我对设置最满意的典范（另一个是 Firefox），打开默认是简单设置，简单设置基本涵盖了所有常用功能，并且基本没有用不到的设置。这个播放器对我来说最大也是惟一一个缺点就是界面。丑，太丑，丑到连 MPC 都不如。。

#PotPlayer
可以讲因为 PotPlayer 的出现终结了我一直换播放器的历程。

#设置
来张集体照（就这6个，其他播放器就不下载装了，脏电脑）
我觉得高下立判—— VLC > QQ影音 > MPC = PotPlayer > MPlayer > 射手 > KMPlayer
QQ影音分类最清晰，但是没有高级设置，PotPlayer 虽然选项很多，但是分类清晰，并且实际操作中基本只需要到第一级菜单里就行。MPlayer 都是在一级里的，KMPlayer 不但有一二三级，并且还有标签页，进一步复杂。射手则是太简单，并且是提供左右按钮来切换设置项，你不知道你想要的东西到底在哪一页。

#界面
还是集体照，不过这已经是我修改过的了，很明显你能看出有个奇葩。。。抛开 KMPlayer 这傻逼不谈，我认为作为一个播放器，应该是给用户最少的干扰，所以尽可能隐藏界面，既只有想要对播放进行操作的时候才显示控制界面，对于这个问题各个播放器也都有解决方案。
上策（PotPlayer、射手影音、QQ影音）：只有鼠标移动到指定区域才显示控制界面。

  • PotPlayer 做得最好，控制按钮很窄，位于顶部和底部，播放器边框默认是深色，看视频时一般都能较好的融合。
  • 射手 次之，他的播放条位于中下位置，可移动，但是还是很碍眼，经常会显示。标题栏和边框特别显眼。
  • QQ影音 最烂，控制按钮占到了近四分之一的屏幕，所以经常会误显示。标题栏和边框特别显眼。
    中策（MPlayer）：移动到位置，界面会有变化，回多一个框和标题栏，并且导致播放器视频无法紧贴屏幕。
    下策(VLC、MPC)：只能只显示播放条或者是控制按钮，不能同时存在。并且界面还是必须显示的，不能隐藏，只是把干扰界面变窄而已。
    PotPlayer 的内置皮肤，是我用过的所有播放器的所有官方和是三方的皮肤里最棒的，将干扰降到最小并且功能齐全。刚找到一款类似感觉，但是是英文，并且感觉有点过于复杂的 PotPlayer 皮肤——Youtube。
    VLC什么都好，就是这个界面难看。第三方皮肤各种BUG。

#快捷键
很显然， QQ影音 和 PotPlayer 完胜，腾讯的用户体验真不是盖的（艹，就是播放器界面做不好）。这几个开源播放器都是什么啊，这么一长条是想闹哪样？很难找到你想要的那个，即使找到了设置也非常麻烦。PotPlayer 是惟一一个在设置里没有把快捷键都列出来的播放器，而是在右键菜单对应的地方表明快捷键。想修改就自己添加。并且 MPC 的键盘默认快捷键非常蛋疼。
鼠标按键 MPC 和射手都是键鼠放一起，显然这是极其二逼的行为，键盘有 100 左右个键，一般的鼠标就三个，并且这两个的快捷键设置框都无法把所有控制项都显示出来。。二逼至极。

#播放列表
做得最好的还是 QQ影音，当你打开一个文件，会自动添加到播放列表的底部，并且播放。（哈哈，这个建议是我提的，我果然对体验有过人的见解）。
PotPlayer 就很烂，当播放器没有打开时，你双击打开文件回清楚你的播放列表，即只有你打开的这一个文件。相当蛋疼，但是我没办法提建议，因为作者是韩国人。。
有的播放器就压根没有播放列表的概念，只有最近打开文件的列表。。

暂时只想到这些。。要是还有别的再说吧，睡觉去。
